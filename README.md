# Emaily-react-app
Sample React-app to collect feedback from your users

## Requirements
- Node.js (LTS or higher);
- MongoDB

## How to run
1. Install Dependencies:

	`npm install`

2. Run Application:

	`npm run dev`

## Technologies Used
- Javascript (ES2015)
- React
- Redux
- Redux-Thunk
- Webpack
- Babel
- Axios
- NPM
- Materialize CSS
- Stripe
- SendGrid
- OAuth
- MongoDB
- Git
